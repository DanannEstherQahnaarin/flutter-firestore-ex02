# flutter_application_mms

이 프로젝트는 게시판 기능을 포함한 Flutter 애플리케이션입니다.
사용자는 Firebase 인증을 통해 로그인할 수 있으며, 게시글의 작성, 수정, 삭제와 같은 게시판 관리 기능을 제공합니다.
주요 기능:
- 회원 인증 및 사용자 관리
- 게시글 등록, 수정, 삭제
- 게시글 목록 및 상세 조회
